## Readme

Uart input output ascii text.

baud rate : 115200

通过串口来设置电源板的参数，开关电源通道。设置状态保存在FLASH 中.


### Commands
Start with "AT", end with "\r\n"

AT -> Head

TAG -> VCCOUT

VALUE -> 0

\r\n -> Tail


1. Ping

```
AT\r\n
OK\r\n
41 54 0D 0A
```

2. Set VCC_OUT

不用

```
AT+VCCOUT?\r\n
41 54 2b 56 43 43 4f 55 54 3f 0D 0A
// 12V
0\r\n 
// 19V
1\r\n 
AT+VCCOUT=0\r\n
41 54 2b 56 43 43 4f 55 54 3d 30 0d 0a
OK\r\n
AT+VCCOUT=1\r\n
41 54 2b 56 43 43 4f 55 54 3d 31 0d 0a
OK\r\n
```

3. Set Echo Mode

不用

```
AT+ECHO?\r\n

41 54 2b 45 43 48 4f 3d 30 0d 0a 
OK\r\n
AT+ECHO=1\r\n
41 54 2b 45 43 48 4f 3d 31 0d 0a 
OK\r\n

```
